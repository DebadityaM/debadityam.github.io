Welcome to my corner of the internet. This is where I'll write about things I'm
learning and building — mostly around AI, machine learning, and backend
development.

## Why a blog

Writing forces me to actually understand a thing rather than just skim it. If a
post here helps someone else too, even better.

## What to expect

- Notes from projects I'm working on
- Things I wish someone had explained to me more clearly
- The occasional deep dive when a topic grabs me

More soon.

## How I add a new post

Each post is a Markdown file in `src/content/posts/`. Create a file like
`my-post.md`, give it frontmatter like the block at the top of this file
(`title`, `pubDatetime`, `description`, `tags`), write in Markdown below it, then
commit and push. The site rebuilds and redeploys automatically.
